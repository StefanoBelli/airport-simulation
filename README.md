# airport-simulation
PMCSN project a.y. 2024/2025

## Assemble JAR and run the project

From the project root:

```bash
$ mvn package
$ java -jar target/simulation-0.0.0-jar-with-dependencies.jar
```

## Cleaning

*This will remove the "target" directory*

From the project root

```bash
$ mvn clean
```
